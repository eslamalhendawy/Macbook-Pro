# 💻 MacBook Pro – Website Replica

This project is a high-fidelity replica of Apple’s MacBook Pro website, focused on recreating the premium product presentation, immersive animations, and smooth interactive experience.

The objective of this project was to practice advanced animation workflows, 3D integration, and building highly polished, performance-focused frontend interfaces.

---

## ✨ Features

- 🎯 **High-Fidelity UI Replica**
  - Carefully recreated layout and visual hierarchy  
  - Pixel-accurate spacing and typography  
  - Smooth section flow and premium presentation  

- 🎬 **Advanced GSAP Animations**
  - Smooth page and section transitions  
  - Scroll-triggered animation effects  
  - Timeline-based motion sequences  
  - Dynamic content reveals  
  - GSAP animations implemented across the entire website  

- 🧊 **Interactive 3D MacBook Model**
  - Fully interactive 3D model integration  
  - Real-time user interaction  
  - Smooth rotation and perspective adjustments  
  - Immersive product showcase experience  

- 📱 **Responsive Design**
  - Optimized for desktop, tablet, and mobile  
  - Smooth layout scaling across screen sizes  

- ⚡ **Performance-Focused Architecture**
  - Built with reusable React components  
  - Clean and maintainable project structure  
  - Optimized rendering and animation performance  

---

## 🛠️ Tech Stack

- **Frontend:** React  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP  
- **3D Rendering:** Three.js (if used)  
- **Build Tool:** Vite

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/eslamalhendawy/macbook-pro.git
cd macbook-pro
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

The app will run locally at:

```
http://localhost:5173
```
